# E-Commerce Review Sentiment Analysis App

This project is a Streamlit web app for NLP-based sentiment analysis of e-commerce customer reviews.

## Features

- Dataset overview
- Sentiment distribution
- Rating distribution
- Review length analysis
- Word cloud
- TF-IDF vectorization
- Multiple ML model training
- Model comparison
- Sentiment prediction for new reviews
- Final model evaluation

## Models Used

- Multinomial Naive Bayes
- Complement Naive Bayes
- Bernoulli Naive Bayes
- Logistic Regression
- SGD Classifier
- Passive Aggressive Classifier
- Ridge Classifier
- Linear SVC
- Random Forest
- Extra Trees
